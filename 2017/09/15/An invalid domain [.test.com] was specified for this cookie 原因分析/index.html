domain规则如下
1、必须是1-9、a-z、A-Z、. 、- （注意是-不是_）这几个字符组成
2、必须是数字或字母开头

上篇文章使用.test.com报错就是因为使用”.”开头

3、必须是数字或字母结尾

path的规则源码及规则

1、字符必须是在 0x20-0x7E之间，并且不能出现”;”号

cookie value 源码及规则

1、会自动去除开头和结尾的引号”
2、如果包含以下规则字符则校验失败：
c < 0x21 || c == 0x22 || c == 0x2c || c == 0x3b || c == 0x5c || c == 0x7f
